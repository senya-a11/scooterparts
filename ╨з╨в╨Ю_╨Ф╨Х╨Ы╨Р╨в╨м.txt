# IMPORT / Fm TuN — v18 FINAL

## Запуск проекта

```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Создайте .env файл (скопируйте из .env.example)
cp .env.example .env
# Заполните DATABASE_URL, SECRET_KEY, ADMIN_PASSWORD и т.д.

# 3. Запустите сервер
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000
```

- users: personal_discount INTEGER, is_manager BOOLEAN
